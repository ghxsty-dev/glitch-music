# GLITCH Music

<img src="https://github.com/ghxsty-dev/glitch-music/blob/main/.github/GLITCHyazi%20(800%20x%20200%20piksel).png?raw=true&" width="1200" />

* GLITCH Music, klasik bir müzik oynatıcıdan çok daha fazlasıdır — tam anlamıyla kişisel bir müzik merkezi gibi çalışır. Yerel müzik kütüphanesini, YouTube Music arama ve indirme özelliklerini, oynatma listesi yönetimini, gelişmiş modern bir oynatıcıyı, senkronize şarkı sözlerini ve sanatçı/albüm keşfini tek bir uygulamada bir araya getirir. Ayrıca tema seçenekleri, şeffaflık modları, performans ayarları, kısayollar ve davranış tercihleri gibi geniş özelleştirme seçenekleri sunar; tüm bunlar akıcı ve masaüstü odaklı bir arayüz ile birleşir.

* GLITCH Music’i diğer oynatıcılardan gerçekten ayıran şey ise ekosistem benzeri deneyimidir. Discord Rich Presence entegrasyonu, gelişmiş uygulama içi indirme ve aktarım sistemleri, aylık dinleme özetleri ve istatistikleri, çevrimdışı öncelikli tasarım anlayışı ve kişiselleştirilmiş öneri/keşif sistemleri sayesinde sadece bir müzik oynatıcı olmaktan çok daha fazlasını sunar. Müzik deneyiminizi düzenlemenize, keşfetmenize ve tamamen kişiselleştirmenize yardımcı olan kapsamlı bir uygulamadır.

## Kurulum

* Uygulamayı GLITCH Music Setup üzerinden indirdikten sonra kurulum adımlarını takip edin. Eğer gerekli modüller eksikse aşağıdaki komutları kullanın:

## Windows

```bash
winget install python
winget install ffmpeg
pip install ytmusicapi
winget install yt-dlp
```

## Linux

```bash
sudo apt update
sudo apt install -y python3 python3-pip ffmpeg
python3 -m pip install --user -U ytmusicapi yt-dlp
```

Linux paketlerini oluştururken proje klasörünü Linux dosya sistemi içinde kullanın. `/mnt/c` üzerinden build almayın:

```bash
npm ci
npm run dist:linux
```

Linux build dosyalarını GitHub Releases üzerinden yayınlamak için:

```bash
npm run dist:linux:github
```

## Nasıl Kullanılır

* Uygulamayı başlattıktan sonra sol üst köşede bulunan Create butonunu kullanarak oynatma listeleri oluşturabilirsiniz. Ayrıca TXT ile İçe Aktar özelliği sayesinde mevcut listelerinizi de ekleyebilirsiniz.

* YouTube Music oynatma listelerini aktarmak için sağ üst köşedeki Settings bölümünü açın, ardından Shared Source sekmesine gidin, Google hesabınızı bağlayın ve oynatma listelerinizi doğrudan içe aktarın.

* Eğer bir hata ile karşılaşırsanız veya bizimle iletişime geçmek isterseniz, aşağıdaki bağlantıya tıklayarak Discord sunucumuza katılabilirsiniz.
